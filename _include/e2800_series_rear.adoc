 a| 
MAC位址

管理連接埠1（「P1」）的MAC位址。如果您使用DHCP取得原始控制器的IP位址、則需要此位址才能連線至新的控制器。

2.
 a| 
FRU零件編號

FRU零件編號。此編號必須符合目前安裝之控制器的更換零件編號。



 a| 
3.
 a| 
4埠HIC
 a| 
4埠主機介面卡（HIC）。執行更換時、必須將此卡移至新的控制器。

*附註*：E2800A控制器沒有HIC。
