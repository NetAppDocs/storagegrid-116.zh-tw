= 設定StorageGRID 鏈接（SG5700）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必須先設定應用裝置與您計畫使用的網路之間的連線、才能將StorageGRID 該應用裝置部署為StorageGRID 位於整個資訊網中的儲存節點。您可以瀏覽StorageGRID 至E5700SG控制器（應用裝置中的運算控制器）上隨附的《SESSE應用裝置安裝程式》來設定網路功能。
