= 變更CIFS稽核共用使用者或群組名稱
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以新增新的使用者或群組、然後刪除舊的使用者或群組、來變更CIFS稽核共用的使用者或群組名稱。

透過CIFS/Samba進行的稽核匯出已過時、將在未來StorageGRID 的版本中移除。

.步驟
. 將新的使用者或群組以更新名稱新增至稽核共用區。
. 刪除舊的使用者或群組名稱。
